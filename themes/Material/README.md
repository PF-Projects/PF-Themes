 - Articles
 - Blog
 - Downloads
 - Error Page
 - Forum
 - Home
 - Messages

 - Light and Dark mode
 - Social Networks Admin
 - Built-in Footer Panels with Excluding
 - Ability to choose Image logo or Text logo in menu
