# Atom9 Ignition Pack
Atom9 is a package of 3 themes with a shared core.

![Version](https://img.shields.io/badge/Version-1.5.1-blue.svg)

Themes:
- DefaultPack
    - Custom templates: Home
- ModFusion
    - Custom templates: Home
- StarCity
    - Custom templates: Downloads, Home, Login / Register, News, Private Messages
